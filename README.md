# Livro Vivo  
## Ecossistema Vibracional da Consciência

### Visão Geral  
O Livro Vivo e o Livro Fonte formam um organismo literário integrado — um **ecossistema vibracional** que explora, em múltiplas camadas, a consciência, a experiência humana e a arquitetura sutil do ser.

Cada livro é uma pétala de um mesmo campo, revelando uma faceta única do sistema:

| Nº | Título                                      | Enfoque                                   | Palavra-chave     |
|----|---------------------------------------------|--------------------------------------------|-------------------|
| 1  | *Atravessando o Campo da Consciência*      | Poético, íntimo, subjetivo — Travessia     | percepção / experiência |
| 2  | *A Arquitetura do Ser*                      | A estrutura interna da consciência         | forma / estrutura |
| 3  | *A Prática do Ser*                          | Aplicado, cotidiano, comportamental        | prática / aplicação |
| 4  | *O Campo Humano*                            | Seres humanos, coletivos, teias            | humano / coletividade |
| 5  | *A Ciência da Luz*                          | Fenomenologia vibracional, física sutil    | luz / fenomenologia |
| 6  | *Corpo Consciente*                          | Encarnação, presença, corpo como campo     | corpo / encarnação |
| —  | *Livro Fonte – A Consciência do Sistema*    | Origem, matriz, o organismo inteiro        | sistema / origem  |

**Essência das camadas**  
- Livro Vivo → percepção  
- Livro Fonte → sistema  
- Arquitetura do Ser → estrutura interna  
- O Campo Humano → coletividade  
- Ciência da Luz → fenomenologia  
- Corpo Consciente → encarnação  
- A Travessia → experiência  
- A Prática → aplicação  

E, no centro, o **Livro Fonte → o Organismo inteiro**.

### Como Contribuir  
1. Faça um _fork_ deste repositório.  
2. Crie uma nova _branch_ para a sua contribuição: `git checkout -b feature/sua-ideia`.  
3. Faça seus ajustes ou adições, e depois faça um _commit_ com mensagem clara.  
4. Abra um _Pull Request_ (PR) com a descrição do que foi feito e como se conecta ao ecossistema vibracional.  
5. Aguarde revisão, estamos construindo este organismo em colaboração.
